# Test case for PERX

React, Redux, AntD

## Local run

1. Clone project
```bash
git clone https://github.com/beloglazof/perx-test-case.git 
```

2. Go to project folder
```bash
cd perx-test-case
```

3. Install dependencies
```bash
yarn
```

4. Start development server
```bash 
yarn start
```

5. Open http://localhost:3000

That's it. You awesome!
